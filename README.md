# 🎮 Hangman Game in Python

A simple console-based Hangman game where the player guesses a word one letter at a time. 
The game is beginner-friendly and demonstrates basic Python concepts like loops, 
conditionals, strings, lists, and user input.



 📌 Features

✅ 5 predefined words (no external files or APIs)
❌ Max 6 incorrect guesses
 🔁 Option to play again after the game ends
🧠 Basic input validation and feedback
💡 Simple and clean terminal-based UI


 🛠️ Tech Used

 Python 3.13
 Core concepts: `random`, `while` loop, `if-else`, `strings`, `lists`


 ▶️ How to Play

1. Clone or download the repository.
2. Run the Python file:

   ```bash
   python hangman.py
